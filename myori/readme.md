### ? 주요 기능

* **사주 엔진:** 사용자의 생년월일시를 입력받아 **사주팔자 및 오행**을 정확히 계산하고 저장합니다냥.

## ?? 3. 개발 환경 설정 및 실행 방법

### **A. 데이터베이스 설정 (필수)**

1.  **MySQL 접속:** `dothome.co.kr` 또는 로컬 환경의 MySQL 데이터베이스에 접속합니다냥.
2.  **스키마 생성:** `/db/schema.sql` 파일을 실행하여 `users`, `diaries`, `analysis_results` 등 모든 테이블을 생성합니다냥.
3.  **데이터 삽입:** `/db/init_manse.sql` 파일을 실행하여 만세력 기준 데이터를 초기 삽입합니다냥.
4.  **연결 정보 수정:** `/modules/db_connect.php` 파일에 **실제 DB 접속 정보**를 정확하게 입력합니다냥.

### **B. 프로젝트 실행**

1.  모든 파일을 호스팅 서버의 `/public_html` 또는 `/www` 폴더에 업로드합니다냥.
2.  웹 브라우저에서 서버 주소로 접속합니다냥. (`http://[혜츄히메주소]/index.php`)
